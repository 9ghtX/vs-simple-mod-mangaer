## Description

A simple mod manager for Vintage Story game that allows you to view installed mods and copy them from a local game installation to a server data directory.

## Download

Download the latest release from the **Releases** page. No installation required. Just run the `.exe` file.

## Usage

1. Select the game installation folder.
2. Select the server data folder.
3. The program will display installed mods.
4. Press **Copy to server**.

## Features

- View installed mods
- Copy mods to server
- Backup support
- Compact / expanded table mode
